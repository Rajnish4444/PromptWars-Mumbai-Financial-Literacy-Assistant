# Product Requirements Document

## Product name
FinLit Lab

## Problem statement
People often find financial concepts like saving, budgeting, and investing confusing because the explanations are too technical, too abstract, or not interactive enough.

## Goal
Help beginners understand core financial concepts in a simple and interactive way.

## What this product is
A web-based financial literacy learning tool with:
- concept explainers
- interactive simulations
- quizzes
- an AI tutor
- progress tracking

## What this product is not
- a personal finance manager
- a portfolio tracker
- a tax tool
- a loan underwriting tool
- a personalized financial advisor

## Target users
- students
- first-time earners
- beginners with little finance knowledge
- users who want simple explanations and practical examples

## User needs
Users need to:
- understand key finance terms
- see how concepts behave in practical situations
- test what they learned
- ask follow-up questions
- know what to learn next

## Core features
1. Learn modules
2. Interactive simulators
3. Quiz and scenario mode
4. AI tutor
5. Learning progress tracking

## Success criteria
- A beginner can understand at least one financial concept in under 2 minutes.
- A user can interact with simulations and explain what changed.
- A user can complete a quiz and understand mistakes.
- A user can ask AI for a simpler explanation and get a safe educational answer.
- The demo clearly shows learning value, not just UI polish.

## Feature acceptance criteria

### Learn
- Each module explains one concept in plain English.
- Each module includes an example and a quick check question.

### Simulate
- Every simulator updates instantly from deterministic logic.
- Every result includes a “what this means” explanation.

### Quiz
- Questions return immediate feedback.
- Explanations clarify correct and incorrect answers.

### AI Tutor
- AI can simplify concepts and explain simulator outputs.
- AI does not replace the app’s calculations.
- AI fallback is handled gracefully.

### Progress
- Tracks completed modules, quiz performance, and weak topics.
- Recommends what to learn next.

## Non-goals
- advanced investing analytics
- personalized financial planning
- bank account aggregation
- real transaction import
- financial product recommendation
