# Rubric Map

## Problem statement alignment
- Learning-first structure: Learn, Simulate, Quiz, AI Tutor, Progress
- Beginner-friendly language
- Educational framing, not planning

## Code quality
- Angular standalone architecture
- feature-based folder structure
- pure TypeScript simulation logic
- strict typing
- shared UI states

## Security
- Firebase Auth
- Firestore rules with per-user ownership
- App Check
- minimal data collection
- no secrets in repo

## Efficiency
- route-level lazy loading
- signals for local derived state
- deterministic client-side simulations
- minimal dependencies

## Testing
- unit tests for simulation logic
- component tests for key pages
- Playwright for end-to-end learning flows
- emulator tests for Firebase security rules

## Accessibility
- semantic layout
- keyboard support
- focus states
- accessible forms
- text/table fallback for charts
- plain language

## Google services usage
- Firebase App Hosting
- Firebase Auth
- Firestore
- Firebase AI Logic
- Firebase App Check
- optional Firebase Analytics
