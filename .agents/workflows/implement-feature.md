Implement one feature slice completely.

Before coding:
1. Review AGENTS.md and SKILLS.md.
2. Inspect the repo.
3. Summarize what already exists.
4. Propose the smallest complete plan.

Then implement the requested feature with:
- production-minded Angular code
- tests
- loading, empty, and error states
- accessible interactions
- beginner-friendly copy

Feature types may include:
- Learn module
- Simulator
- Quiz
- AI Tutor
- Progress page

Rules:
- keep logic modular
- keep calculations deterministic
- do not introduce unnecessary dependencies
- do not refactor unrelated files

After coding:
- run lint
- run tests
- run build
- summarize changed files, commands run, and remaining risks
